### 深度学习 VS 知识图谱

|          | 深度学习                                   | 知识图谱                                   |
| -------- | ------------------------------------------ | ------------------------------------------ |
| 智能原理 | 对人类大脑的隐性模拟                       | 显性模拟                                   |
| 场景     | 感知智能领域特定任务 如围棋 语音 图像 视频 | 搜索 人机交互                              |
| 特点     | 海量训练数据 强大算力 难以解释             | 海量知识 可解释 可理解                     |
| 进展     | 某些特定任务接近甚至超越人类               | 在知识量上超越人类但是在知识推理上不及人类 |
| 趋势     | 深度融合                                   | 深度融合                                   |

### 知识图谱 VS 传统知识库 VS 数据库

|            | 语义层     | 数据层     |
| ---------- | ---------- | ---------- |
| 关系数据库 | 没有语义   | 丰富的数据 |
| 传统知识库 | 丰富的语义 | 少量的实例 |
| 知识图谱   | 少量的语义 | 丰富的实例 |

#### 优势

知识图谱以传统知识库为基础 但是更注重数据 虽然不能替代数据库 但是能互相补充 且包含一定的语义信息以支持简单的推理  形式灵活可拓展性好

#### 缺陷

缺乏直接的有效处理工具 大规模图谱数据的处理需要借助数据库技术

### 知识图谱应用

| 视角       | 用途                               |

### 语义网络

#### 构成

节点和边构成

节点-表示各种事物、概念、情况、属性、动作、状态等，每个节点可以带有若干属性

边-表示各种语义联系，指明他所连接的节点间某种语义关系

#### 优点

1. 表示自然 易于理解 应用广泛
2. 符合人类联想记忆
3. 结构化知识表示

#### 缺陷

1. 不严格 没有公认的逻辑基础
2. 很难有效处理 灵活但是处理和检索的低效性
